DESCRIÇÃO DO PROJETO

PROJETO DE DESENVOLVIMENTO DE UM SISTEMA DE CADASTRO PARA CASA DE EVENTOS DE MUSICA ELETRONICA

Introdução

Diante dos eventos realizados e alta freqüência dos sócios evolvidos na participação do Clube Park House, faz necessário o controle de dados médicos personalizados de cada pessoa que se tornou sócio.

Objetivo

O projeto em si será desenvolvido para facilitar o trabalho de casas de eventos de musica eletronica e facilitar o trabalho de paramedicos que estao de plantao na festa atravez do sistema que sera desenvolvido.

Tema

O tema proposto visa agregar mais segurança e proteção a saúde dos freqüentadores das festas do clube e afim de facilitar o trabalho dos médicos de primeiros socorros. um sistema de cadastro para o clube ir auxiliar na diminuição de pessoas que passam mal nas festas realizadas pelo mesmo. Para garantir um nível de aceitação dos clientes e profissionais, o sistema possuirá cadastramento de senha onde apenas os médicos e o clube terão acesso aos dados, pela segurança da informação.

Prototipos

Tela inicial com espaço para login utilizando usuário e senha. Com opção "Esqueci minha senha"

Após o login é possivel vizualizar os clientes cadastrados no banco de dados do clube e evenualmente efetuar a consulta de qualquer perfil do cliente que o medico deseja analisar.
O usuario tambem tem a opção de ir para a tela de cadastro do usuario, utulizando o botão "Cadastra Novo Cliente".
Ou podera clicar em "Consulta" que será direcionado a um perfil de um cliente ja cadastrado.

Após isso, o usuario tera todas a opções que deseja para efetuar o cadastro de um novo cliente, nesta tela ele terá que prencher um cadastro basico em relaçao a indentificação e contato do mesmo.

Após o usuario criar o cadastro de contato ele será direcionado para uma tela de cadastro relacionado a saude do novo cliente que será cadastrado.

Após o usuario efetuar todos os passos de cadastro ele será direcionado a esta tela, que informa todos os dados do cliente ja registrado.
